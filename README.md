# PM Club NSUT – Homepage

This is a simple static homepage for **PM Club NSUT** built using only HTML and CSS.  
It follows the task requirements: a navbar, hero section with logo, about section, one extra section, and a footer.

## Features

- Top navigation bar with links to:
  - Home
  - About
  - Events
  - Contact
- Hero section with:
  - PM Club NSUT logo in the center
  - Main heading
  - 1–2 line tagline
  - Call‑to‑action button (Apply Now)
- About section with a short description of the club
- Extra section (**Why Join Us**) describing benefits of the club
- Footer with dummy email and Instagram handle

## Technologies Used

- HTML5
- CSS3
